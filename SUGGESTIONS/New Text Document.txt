PROJECT NAME: PAYROZ B2C APP

PAYROZ will be a pure B2C customer app only. No B2B, no retailer, no distributor, no AEPS, no agent system.

MAIN REQUIREMENT:
Build a top-level India fintech B2C app like Amazon Pay / PhonePe style, but unique PAYROZ design.

WALLET RULE:
1. No Add Money Wallet
2. No user wallet top-up
3. Only Rewards Wallet
4. Rewards Wallet balance will include:
   - Cashback
   - Referral Income
   - Failed Transaction Refund
   - Admin Promotional Bonus
5. User can use Rewards Wallet balance for eligible services.
6. Direct payment gateway payment for all services.
